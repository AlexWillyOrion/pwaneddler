# pwaneddler
A script kiddie for perform MiTM attack and replace image in a host's browser or for turn off a host's internet connection

#dependencies
- Ettercap (and Etterfilter)
- Nmap
- Php
 

For use: 
```
chmod +x pwaneddler.php
./pwaneddler.php
```
